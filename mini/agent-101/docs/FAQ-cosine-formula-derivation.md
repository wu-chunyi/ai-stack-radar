# 余弦公式推导 · 点积的几何含义 · 为什么除以长度

---

## 点积的几何含义（核心）

"分量相乘再累加"在几何上等于：

```
A · B = |A| × |B| × cos(θ)

其中：
  |A| = A 向量的长度
  |B| = B 向量的长度
  θ   = 两向量之间的夹角
```

**物理意义**：点积 = A 在 B 方向上的投影长度 × B 的长度

```
         B
       ↗
     ↗  ↑
   ↗    | ← A 在 B 上的投影 = |A|cos(θ)
 ↗──────┘
●──────────→ A
```

---

## 为什么分量相乘累加 = |A||B|cos(θ)（数学推导）

以 2D 为例（1024D 同理）：

```
A = [a₁, a₂]    → a₁ = |A|cos(α)，a₂ = |A|sin(α)
B = [b₁, b₂]    → b₁ = |B|cos(β)，b₂ = |B|sin(β)

分量点积 = a₁b₁ + a₂b₂
         = |A|cos(α)·|B|cos(β) + |A|sin(α)·|B|sin(β)
         = |A||B| · (cos(α)cos(β) + sin(α)sin(β))
         = |A||B| · cos(α - β)    ← 三角恒等式
         = |A||B| · cos(θ)        ← θ = 两向量夹角
```

代入数字验证（A=[4,1]，B=[2,3]）：
```
分量点积 = 4×2 + 1×3 = 11
几何公式 = √17 × √13 × cos(42°) ≈ 4.12 × 3.61 × 0.74 ≈ 11 ✅
```

## 余弦公式的推导（一步代数）

已知 `点积 = |A| × |B| × cos(θ)`，两边除以 `|A|×|B|`：

```
        点积
cos(θ) = ─────────    ← 余弦公式从这里来，不是"发明"，是"解出来"的
        |A| × |B|
```

## 为什么"除以长度"能消除长度影响

```
A = [1, 0]     ← 短向量，指右
B = [100, 0]   ← 长向量，也指右（完全同向）
C = [0, 1]     ← 短向量，指上（垂直）

点积（不除长度）：
  A·B = 100（很大，但因为 B 长，不反映真实方向）
  A·C = 0  （正确）

余弦（除以长度）：
  cos(A,B) = 100 / (1×100) = 1.0 ← 完全同向 ✅
  cos(A,C) = 0   / (1×1)   = 0.0 ← 垂直=无关 ✅
```

除以长度 = 把两个向量都"缩短到长度为 1"，然后只比较方向。

## 代码与公式的完整对照

```ts
dot  += a[i] * b[i]    // 分量相乘累加 = 点积 = |A||B|cos(θ)
magA += a[i] * a[i]    // a[i]² 累加   = |A|²
magB += b[i] * b[i]    // b[i]² 累加   = |B|²
dot / (√magA × √magB)  // |A||B|cos(θ) / |A||B| = cos(θ)
```

最后一行的约分：
```
|A| × |B| × cos(θ)
────────────────────  =  cos(θ)
     |A| × |B|
```

## 三个问题的答案

```
为什么用点积？
  → "分量相乘累加"在几何上就等于 |A||B|cos(θ)
    自然把长度和角度都编码进去了

为什么除以 |A|×|B|？
  → 消除向量长度的影响，只留下 cos(θ)（纯方向比较）

余弦公式从哪来？
  → 从 A·B = |A||B|cos(θ) 这个几何事实
    两边除以 |A||B| 直接推出，不是规定，是推导
```
