# Dify 评测

## Dify

### 简介
Dify 是一个开源的 LLM 应用开发平台，提供从数据管理到模型部署的一站式解决方案。它强调 RAG（检索增强生成）和提示词工程，支持可视化编排和 API 发布。

### 核心功能
1. **知识库管理**：支持上传 PDF、网页、数据库等 10+ 格式，自动分块和向量化（支持 Pinecone、Weaviate 等）。
2. **提示词工作流**：可视化编排多步骤提示，支持变量、条件分支和循环。
3. **内置评估**：提供 A/B 测试和人工反馈系统，可对比不同模型/提示的效果。
4. **多租户与权限**：支持团队协作，可设置角色（管理员、开发者、测试员）。

### 定价
Freemium。开源版免费（Apache 2.0 许可证），云版（Dify Cloud）起价 $49/月（含 5000 次 API 调用和 10GB 存储）。

### 优点
1. **RAG 集成深度**：知识库管理功能强大，自动分块和检索优化效果好。
2. **评估系统完善**：内置测试工具，减少部署后调试成本。
3. **部署灵活**：支持自托管（Docker）和云版，适合不同规模团队。

### 缺点
1. **学习成本高**：界面功能繁多，新手需花时间熟悉模块。
2. **性能依赖配置**：自托管时需手动优化向量数据库和模型推理延迟。

### 适用场景
企业级 RAG 应用（如智能客服、内部知识问答）、提示词实验和模型评估平台。
