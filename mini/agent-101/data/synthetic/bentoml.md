# BentoML 评测

## BentoML 评测

### 简介
BentoML 是一个开源模型部署框架，专注于简化从训练到生产的流程，支持 PyTorch、TensorFlow、Hugging Face 等框架，提供自动容器化和 Kubernetes 集成，适合快速原型开发。

### 核心功能
1. **一键部署**：通过 `bentoml serve` 命令启动 API 服务器，自动生成 Docker 镜像，打包时间 < 1 分钟。
2. **多框架集成**：原生支持 Hugging Face、Scikit-learn、XGBoost，可加载 100+ 预训练模型。
3. **自动缩放**：集成 Kubernetes 和 Ray，支持基于 CPU/内存的自动缩放，并发处理 500+ 请求。
4. **模型管理**：提供模型注册表，支持版本控制和回滚，存储 50+ 模型版本。
5. **监控和跟踪**：集成 OpenTelemetry，提供请求延迟和错误率仪表盘。

### 定价
免费（开源，Apache 2.0 许可证），BentoCloud 托管服务起价 $49/月（含 100 小时 GPU 使用）。

### 优点
1. **快速部署**：从模型到 API 只需 5-10 分钟，适合实验和原型开发。
2. **轻量级**：基础镜像仅 200MB，启动时间 < 5秒，适合边缘设备。
3. **社区活跃**：GitHub 星标 7k+，每周更新，插件生态丰富（如 AWS、GCP 集成）。

### 缺点
1. **性能瓶颈**：对于大型 LLM（如 70B 参数），推理延迟较 vLLM 高 30-50%。
2. **文档不够详细**：高级功能（如分布式部署）示例少，调试困难。

### 适用场景
快速原型开发和中小规模部署，如内部工具、Demo 应用、教育项目，适合团队快速验证想法。
