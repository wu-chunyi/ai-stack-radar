# LangGraph 评测

## LangGraph

### 简介
LangGraph 是一个用于构建有状态、多参与者 AI 工作流的框架，基于 LangChain 生态系统。它允许开发者将 LLM 调用、工具使用和条件逻辑编排为可循环执行的图结构，特别适合复杂的 Agent 和对话系统。

### 核心功能
1. **有状态图结构**：支持循环、分支和并行节点，可处理多轮对话中的状态持久化。
2. **内置支持人类介入**：允许在关键节点暂停流程，等待人工反馈或审批。
3. **流式与异步执行**：支持节点级别的流式输出（如 token 流）和异步运行，提升实时交互体验。
4. **与 LangChain 无缝集成**：可直接使用 LangChain 的模型、工具和记忆组件，降低学习成本。

### 定价
Free + 开源（MIT 许可证）。LangSmith 平台提供付费追踪和调试（按使用量计费，约 $0.01/1000 次调用）。

### 优点
1. **高度灵活**：能构建任意复杂度的 Agent 流程，如循环搜索、多步骤推理。
2. **状态管理出色**：自动处理上下文传递，无需手动维护变量。
3. **社区活跃**：文档详尽，示例丰富，更新频繁（每月发布 2-3 次新版本）。

### 缺点
1. **学习曲线陡峭**：图的概念和节点配置较为抽象，新手需投入时间理解。
2. **调试困难**：复杂图结构出错时，追踪问题点需要额外工具（如 LangSmith）。

### 适用场景
构建需要多轮推理、工具调用和条件分支的 Agent（如客服系统、自动化研究助手），尤其适合生产级复杂工作流。
