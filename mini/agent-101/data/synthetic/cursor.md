# Cursor 评测

## Cursor

### 简介
Cursor 是一款基于 VS Code 深度定制的 AI 代码编辑器，内嵌了 GPT-4、Claude 3.5 等大模型，专为加速开发流程而设计。它通过对话式编程、代码补全和上下文理解，帮助开发者减少重复劳动，尤其适合日常编码和项目重构。

### 核心功能
- **AI 对话式编程**：在侧边栏或内联对话框中，可针对选中代码提问、修改或生成新代码，支持多轮上下文。
- **智能代码补全**：基于项目上下文（如变量名、函数返回类型）提供单行或多行补全，准确率约 85%。
- **代码库索引**：自动索引整个项目，支持跨文件引用，如“添加一个登录函数，使用已有的数据库连接”。
- **一键重构**：支持重命名、提取函数、优化导入等操作，通过 AI 建议减少手动改动。

### 定价
- **Free**：基础 AI 补全和 200 次/月的对话。
- **Pro**：$20/月，包含无限对话、GPT-4 访问和代码库索引。
- **Business**：$40/月，增加团队管理和自定义模型。

### 优点
- **深度集成 VS Code**：保留原版快捷键、扩展生态系统，迁移成本极低。
- **上下文敏感**：能理解项目结构，生成的代码更贴合实际，例如自动匹配现有 API 风格。
- **速度快**：补全延迟低于 500 毫秒，日常使用无阻塞感。

### 缺点
- **资源占用较高**：索引大型项目（如 10 万行）时，内存占用可达 1GB。
- **依赖网络**：离线模式功能有限，仅支持基本语法提示。

### 适用场景
适合前端、后端开发者进行日常编码、重构和快速原型开发，尤其是需要频繁切换上下文的项目。
