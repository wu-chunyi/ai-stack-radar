# Codegen 评测

## Aider

## 简介
Aider 是一个基于终端的 AI 编程助手，专注于通过自然语言对话直接编辑本地代码文件。它由 Paul Gauthier 开发，支持多种语言和框架，特别强调与 Git 版本控制的深度集成。

## 核心功能
- **多文件同步编辑**：支持同时修改多个文件，并能理解文件间的依赖关系，例如自动更新函数调用签名。
- **Git 集成**：每次修改前自动创建提交点（commit），允许用户轻松回滚错误更改。
- **代码库映射**：自动生成代码库的树状结构，帮助 AI 理解项目上下文，减少幻觉。
- **多种模型切换**：支持 GPT-4、Claude 3 Opus、Llama 3 等，且能通过 `--model` 参数即时切换。

## 定价
- **Free**：开源项目，通过 `pip install aider-chat` 安装。用户仅需支付模型调用费用（如 GPT-4 约 $0.03/1K tokens 输入）。

## 优点
1. **无缝集成**：直接在终端工作，无需离开编辑器，适合习惯命令行的开发者。
2. **安全回滚**：Git 自动提交机制让每次 AI 修改都留有备份，极大降低了风险。
3. **上下文感知**：能自动读取项目配置文件（如 `requirements.txt`），提供更符合项目规范的代码。

## 缺点
1. **学习曲线**：需要用户熟悉终端操作和 Git 基础，对新手不够友好。
2. **大项目性能**：对于超过 10 万行代码的大型项目，代码库映射加载时间较长，影响体验。

## 适用场景
- 个人开发者快速实现功能原型或重构代码片段。
- 团队在代码审查前，使用 Aider 进行自动化编码建议。
- 教育场景中，辅助学生理解代码逻辑和修改后果。
