# MaxKB 评测

## MaxKB

### 简介
MaxKB 是一个开源的知识库问答工具，由 1Panel 团队开发，专注于快速搭建基于私有数据的 AI 问答系统。它提供 Docker 一键部署，支持多种模型，适合中小企业和个人开发者。

### 核心功能
- **知识库管理**：支持上传 PDF、Markdown、文本文件，自动分段并向量化，处理速度 50 页/分钟。
- **多模型支持**：兼容 OpenAI、Qwen、DeepSeek 等模型，可用本地部署的 Ollama。
- **对话历史**：保存用户交互记录，支持回溯和导出，便于分析。
- **API 接口**：提供简单 REST API，可嵌入网站或小程序。

### 定价
- **Free**：开源版本，完全免费，包含所有核心功能。
- **Freemium**：云服务版免费层（20 次查询/天），付费版 $9.9/月（1,000 次查询，含自动更新）。
- **Paid**：企业版 $49.9/月，提供私有部署和专属支持。

### 优点
- 部署极简：一行命令 `docker run` 即可启动，适合新手。
- 轻量级：资源占用低（2 GB RAM 即可运行），适合低配服务器。
- 易扩展：支持自定义提示词和模型切换，灵活性不错。

### 缺点
- 功能单一：缺乏复杂工作流和高级分析，仅限问答。
- 文档解析弱：不支持图片和表格，仅处理纯文本。

### 适用场景
适合个人博客或小型团队，如用 MaxKB 搭建技术文档问答系统，或快速测试 RAG 概念验证。
