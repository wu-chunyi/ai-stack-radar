# Continue 评测

## 简介
Continue 是一款开源 AI 代码辅助工具，专注于在 IDE 中提供上下文感知的交互。它支持 VS Code 和 JetBrains，允许开发者自定义模型或使用本地 LLM，强调灵活性和隐私。作为开源项目，它活跃于 GitHub，社区贡献频繁。

## 核心功能
1. **上下文聊天**：在 IDE 内与 AI 对话，自动引用当前文件或代码块，响应时间约 3-5 秒（基于本地模型）。
2. **代码编辑**：通过自然语言指令生成或修改代码，支持 Python、TypeScript 等，准确率约 80%。
3. **模型自定义**：可接入 OpenAI、Anthropic 或本地模型（如 Llama 2），提供 API 密钥即可。
4. **多文件支持**：能分析整个项目结构，而非单文件，增强上下文理解。

## 定价
Free 模式。完全开源，无需付费；但使用云端 API（如 OpenAI）需自行承担费用。

## 优点
1. **开源透明**：代码可审计，社区驱动更新频繁。
2. **隐私控制**：支持本地模型，代码不离开设备。
3. **高度可定制**：可自由选择模型和配置，适合高级用户。

## 缺点
1. **设置复杂**：需配置 API 或模型，新手入门门槛高。
2. **性能依赖硬件**：本地模型需要高端 GPU，否则响应慢。

## 适用场景
适合注重隐私和可定制性的开发者，尤其是开源贡献者或企业安全团队。
