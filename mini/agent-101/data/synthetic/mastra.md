# Mastra 评测

## Mastra

### 简介
Mastra 是一个专注于 Agent 工作流编排的开源框架，强调低代码和可视化。它允许开发者通过配置文件和 GUI 创建多步骤 AI 流程，支持集成 LLM、API 和自定义脚本。

### 核心功能
1. **可视化工作流编辑器**：通过拖拽节点（如 LLM 调用、条件判断、HTTP 请求）构建流程，实时预览。
2. **多模型支持**：内置 OpenAI、Claude、Gemini 等 5+ 模型适配器，可混合使用。
3. **数据管道集成**：支持从数据库（PostgreSQL、MongoDB）或 S3 加载数据，作为流程输入。
4. **错误重试机制**：节点失败可自动重试（最多 5 次），并记录日志。

### 定价
Freemium。开源版免费（MIT 许可证），云版（托管+协作）起价 $29/月（含 1000 次执行）。

### 优点
1. **低代码友好**：非技术人员可通过可视化界面快速创建简单工作流。
2. **灵活性平衡**：既支持可视化，也允许 YAML 配置和自定义代码扩展。
3. **内置监控**：提供执行历史、延迟和错误统计，方便优化。

### 缺点
1. **性能瓶颈**：复杂工作流（>50 节点）时，可视化编辑器响应变慢。
2. **社区较小**：文档和示例相对较少，遇到问题需自行摸索。

### 适用场景
中小团队快速构建简单的 AI 自动化流程（如邮件摘要、客服路由、数据分类），适合原型验证和内部工具。
