# AnythingLLM 评测

## 简介
AnythingLLM 是一款开源的桌面端应用，旨在将本地运行的 LLM（通过 Ollama、LM Studio 等后端）与文档管理、聊天界面结合，实现私有化 AI 知识库。它支持多种模型（如 Llama、Mistral）和文档格式（PDF、TXT、CSV），默认使用 RAG（检索增强生成）技术，适合需要本地数据安全的个人或团队。

## 核心功能
- **多模型支持**：兼容 Ollama、LM Studio、OpenAI API 等后端，用户可自由切换模型。
- **文档智能处理**：上传 PDF、Word、Markdown 等文件，自动解析并构建向量索引，支持关键词搜索与语义查询。
- **自定义工作区**：创建独立工作区，每个工作区绑定特定文档库，实现隔离化管理（如“项目A”和“财务报告”分开）。
- **对话历史与导出**：保存所有聊天记录，支持导出为 JSON 或 CSV 格式，便于审计或迁移。
- **API 接口**：提供 REST API，可集成到外部应用（如 Slack 或自动化脚本）。

## 定价
**Free**：开源免费，无隐藏费用。但需自行承担硬件成本（如 GPU 或 RAM 消耗），或通过 Ollama 等后端运行模型。

## 优点
1. **隐私优先**：所有数据本地存储，无需上传云端，适合处理敏感文档（如法律合同、医疗记录）。
2. **灵活部署**：支持 Windows、macOS、Linux 三大平台，且可搭配不同后端（本地或自托管）。
3. **文档解析强大**：支持多种格式，且自动分块（chunk size 可调，默认 512 tokens），提升检索精度。

## 缺点
1. **依赖后端**：本身不运行模型，需额外安装 Ollama 或 LM Studio，增加配置复杂度。
2. **资源消耗高**：处理大文档（如 1000 页 PDF）时，内存占用可达 8GB+，低配设备可能卡顿。

## 适用场景
适合企业内网的知识库搭建（如内部 FAQ、员工手册查询），或隐私敏感行业（如法律、医疗）的文档分析。
