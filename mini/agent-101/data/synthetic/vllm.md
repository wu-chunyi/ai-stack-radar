# vLLM 评测

## vLLM 评测

### 简介
vLLM 是一个开源的高性能 LLM 推理引擎，由加州大学伯克利分校开发，专注于优化大语言模型的推理速度和内存管理。它通过创新的 PagedAttention 技术，显著提升了 GPU 利用率，适用于生产环境下的实时推理任务。

### 核心功能
1. **PagedAttention 内存管理**：将 KV 缓存分页管理，减少显存碎片，支持高达 2-4 倍的吞吐量提升，相比 Hugging Face Transformers 实现 24 倍加速。
2. **连续批处理**：动态合并请求，无需等待固定批次完成，支持每秒处理 1500+ 个 token（基于 A100 GPU）。
3. **多模型支持**：兼容 GPT-2、LLaMA、Mistral 等 20+ 主流模型，通过 vLLM Model Zoo 轻松加载。
4. **量化推理**：支持 INT8 和 FP8 量化，降低显存需求 50%，同时保持 95% 以上的模型精度。
5. **OpenAI API 兼容**：提供与 OpenAI 兼容的 REST API，支持流式输出和批量推理。

### 定价
免费（开源，MIT 许可证），无隐藏费用，但需自行承担 GPU 硬件成本（如 A100 约 $3-4/小时云服务）。

### 优点
1. **极致性能**：通过 PagedAttention 实现 2-4 倍吞吐量提升，显存占用降低 60%，适合高并发场景。
2. **易用性强**：单行命令即可部署（`pip install vllm`），集成 OpenAI API 格式，学习成本低。
3. **活跃社区**：GitHub 星标 15k+，每月发布 3-4 次更新，修复速度快。

### 缺点
1. **依赖 CUDA**：仅支持 NVIDIA GPU，对 AMD 或 Apple Silicon 不友好。
2. **模型兼容性**：部分自定义或小众模型需手动适配，文档覆盖不全。

### 适用场景
高吞吐量 LLM 推理服务，如聊天机器人、代码生成 API、实时翻译，尤其适合预算有限的团队使用开源方案。
