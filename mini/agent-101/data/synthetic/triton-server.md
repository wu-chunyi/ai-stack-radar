# Triton Server 评测

## Triton Server 评测

### 简介
Triton Inference Server 是 NVIDIA 推出的开源推理服务器，支持多种框架（PyTorch、TensorFlow、ONNX）和硬件（GPU、CPU），提供模型管理、动态批处理和监控功能，是生产环境的标准部署工具。

### 核心功能
1. **多框架支持**：同时部署 PyTorch、TensorRT、ONNX 模型，通过统一 API 管理，支持 10+ 框架。
2. **动态批处理**：自动合并请求，延迟增加 10ms 但吞吐量提升 3-5 倍（测试于 1000 并发请求）。
3. **模型版本控制**：支持 A/B 测试和回滚，通过 YAML 配置管理 20+ 模型版本。
4. **GPU 调度**：自动分配 GPU 资源，支持多 GPU 负载均衡，利用率达 90% 以上。
5. **监控和日志**：集成 Prometheus 和 Grafana，提供每秒请求数（QPS）、延迟等指标。

### 定价
免费（开源，BSD-3-Clause 许可证），企业级支持需购买 NVIDIA AI Enterprise（约 $10,000/年/节点）。

### 优点
1. **灵活性高**：支持多种框架和硬件，可混用 GPU/CPU，适合异构环境。
2. **生产级功能**：内置负载均衡、健康检查和自动缩放，减少运维成本。
3. **社区成熟**：GitHub 星标 8k+，文档详尽，企业案例丰富（如 Uber、Salesforce）。

### 缺点
1. **性能优化有限**：相比专用推理引擎（如 vLLM），在 LLM 场景下吞吐量低 20-30%。
2. **配置复杂**：YAML 配置文件和模型仓库设置对新手不友好。

### 适用场景
多模型混合部署，如推荐系统、计算机视觉 + NLP 联合推理，适合需要统一管理多种模型的企业。
