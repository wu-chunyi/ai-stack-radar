# TensorRT-LLM 评测

## TensorRT-LLM 评测

### 简介
TensorRT-LLM 是 NVIDIA 推出的 LLM 推理优化库，基于 TensorRT 框架，专为 NVIDIA GPU 设计，提供模型编译、量化、运行时优化，适用于超大规模生产部署。

### 核心功能
1. **图优化编译器**：将模型编译为 TensorRT 引擎，自动融合算子，推理延迟降低 50-80%（如 LLaMA-70B 从 200ms 降至 60ms）。
2. **多精度量化**：支持 FP8、INT4、INT8 量化，INT4 量化后显存减少 75%，吞吐量提升 4 倍（测试于 H100 GPU）。
3. **动态形状输入**：处理可变长度序列，无需填充，节省 30% 计算资源。
4. **多 GPU 并行**：支持张量并行和流水线并行，可扩展至 8 卡 A100 集群，吞吐量线性增长。
5. **集成 Triton Server**：与 Triton Inference Server 无缝集成，提供高性能推理端点。

### 定价
免费（开源，Apache 2.0 许可证），但需 NVIDIA GPU（如 H100 约 $8-12/小时云服务）。

### 优点
1. **极致性能**：编译后推理速度比 PyTorch 快 5-10 倍，适合对延迟敏感的实时应用。
2. **NVIDIA 生态**：与 CUDA、TensorRT、Triton 深度集成，企业级支持。
3. **量化灵活性**：支持多种量化策略，平衡精度和速度，已适配 50+ 主流模型。

### 缺点
1. **GPU 绑定**：仅支持 NVIDIA GPU，AMD 或 Intel 硬件无法使用。
2. **部署复杂**：编译过程需 30-60 分钟，且需熟悉 TensorRT 配置，学习曲线陡峭。

### 适用场景
企业级高性能推理，如自动驾驶语言模型、金融实时交易分析、大规模对话系统，适合拥有 NVIDIA GPU 集群的团队。
