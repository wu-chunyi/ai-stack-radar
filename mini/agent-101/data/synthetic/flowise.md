# Flowise 评测

## Flowise

### 简介
Flowise 是一个开源的、低代码的 LLM 应用构建工具，专注于通过拖拽界面快速创建聊天机器人和 Agent。它支持自定义模型、工具和记忆组件，无需编写代码即可部署。

### 核心功能
1. **拖拽式节点编辑**：提供 100+ 预置节点（如 LLM、提示、记忆、工具），可自由连接。
2. **多模型集成**：支持 OpenAI、Llama 2、Mistral 等 20+ 模型，可本地部署（如 Ollama）。
3. **聊天历史管理**：内置内存节点（Buffer、Summary、Vector Store），支持多轮对话。
4. **一键部署**：生成 REST API 或嵌入 iframe，支持 Docker 和 Railway 等平台。

### 定价
Free（开源，Apache 2.0 许可证）。无官方云版，但可通过第三方平台（如 Railway）托管。

### 优点
1. **极低门槛**：无需编程即可构建基础聊天机器人，5 分钟上手。
2. **社区生态丰富**：用户贡献大量模板和节点，可直接复用。
3. **本地部署友好**：支持 Ollama 等本地模型，适合数据敏感场景。

### 缺点
1. **功能局限**：复杂工作流（如多 Agent 协作）难以实现，节点灵活性有限。
2. **性能优化困难**：无内置监控或缓存，高并发场景需自行配置。

### 适用场景
快速原型验证、个人或小团队构建简单聊天机器人（如 FAQ 助手、文档问答），适合非技术人员。
