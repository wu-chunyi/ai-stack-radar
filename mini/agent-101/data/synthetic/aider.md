# Aider 评测

## SWE-Agent

## 简介
SWE-Agent 是一个由普林斯顿大学团队开发的开源 AI 智能体，专为自动修复 GitHub 仓库中的软件缺陷而设计。它采用“Agent-Computer Interface”（ACI）架构，能够自主导航代码库、定位问题并生成修复补丁。

## 核心功能
- **自动化缺陷定位**：智能体可以遍历代码目录结构，理解函数调用关系，精准定位 bug 所在文件和行号。
- **补丁生成与提交**：根据分析结果，自动生成符合项目规范的 git diff 补丁，并可直接创建 Pull Request。
- **多模型支持**：兼容 GPT-4、Claude 3 Opus 等主流大语言模型，用户可根据成本与性能灵活切换。
- **SWE-Bench 验证**：在 SWE-Bench 基准测试中，SWE-Agent 的修复成功率达到 12.3%（GPT-4 驱动），显著高于早期基线。

## 定价
- **Free**：开源项目，完全免费使用。用户只需自行配置 API 密钥（如 OpenAI API），按 token 消耗付费。

## 优点
1. **专注性强**：针对代码修复场景深度优化，避免了通用 AI 助手的冗余对话。
2. **可复现性高**：每次运行会生成详细的决策日志和中间状态，便于开发者审查。
3. **社区活跃**：GitHub 星标超过 8000，文档完善，Issue 响应迅速。

## 缺点
1. **依赖外部 API**：用户必须拥有 API 密钥，且复杂任务可能消耗大量 token，成本不易控制。
2. **环境适配有限**：目前主要支持 Python 项目，对于 C++ 或 Java 等语言的支持尚在实验阶段。

## 适用场景
- 开源项目维护者，希望自动化处理 Issue 中的 bug 报告。
- 团队在 CI/CD 流程中集成自动修复工具，减少人工排查时间。
- 研究人员研究 AI 在软件工程中的自主决策能力。
