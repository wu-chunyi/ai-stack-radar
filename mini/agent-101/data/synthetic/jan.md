# Jan 评测

## 简介
Jan 是一款开源的桌面端 AI 助手，允许用户本地运行 LLM（如 Llama、Mistral、Gemma），并集成图像生成（通过 Stable Diffusion）和语音转文字功能。它强调隐私和离线使用，提供类似 ChatGPT 的界面，但所有数据保留在本地。

## 核心功能
- **多模态支持**：同时运行文本模型（如 Llama 3）和图像生成模型（如 SDXL），支持文本到图像、图像描述等任务。
- **本地语音输入**：集成 Whisper 模型，支持语音转文字（如中文、英文），无需联网。
- **可扩展插件**：提供插件市场（如 Web Search、PDF Reader），用户可自定义功能。
- **隐私模式**：默认禁止联网，所有请求仅在本地处理，适合敏感场景。
- **模型管理**：支持从 Hugging Face 下载 GGUF 模型，或导入本地文件，并自动分配 GPU/CPU。

## 定价
**Free**：完全开源免费，无付费层级。但运行多模态功能（如 SDXL）需要高端 GPU（如 RTX 3060 12GB）。

## 优点
1. **多模态集成**：在一个界面中同时处理文本、图像、语音，减少工具切换。
2. **离线优先**：无需互联网即可工作，适合偏远地区或网络受限环境。
3. **插件生态活跃**：社区贡献的插件（如 Web Scraper、Markdown 渲染）扩展了实用性。

## 缺点
1. **性能瓶颈**：同时运行文本和图像模型时，GPU 显存占用高（如 8GB VRAM 下 SDXL 生成图像需 30 秒以上）。
2. **安装复杂**：首次配置需要手动安装 Python 依赖（如 PyTorch），对新手不友好。

## 适用场景
适合需要离线多模态能力的用户（如设计师、研究者），或注重隐私的全能型助手。
