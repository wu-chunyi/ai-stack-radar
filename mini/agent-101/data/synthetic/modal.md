# Modal 评测

## Modal 评测

### 简介
Modal 是一个基于云的服务器less 平台，专为 AI 和机器学习工作负载设计，支持 Python 代码直接部署，提供自动缩放 GPU/CPU 资源，免去基础设施管理，适合按需计算任务。

### 核心功能
1. **无服务器推理**：部署模型后自动缩放，从 0 到 1000 并发请求仅需 2 秒，按毫秒计费。
2. **GPU 即用**：支持 A100、H100 等 GPU，按秒计费（A100 约 $0.03/秒），无预留成本。
3. **调度器任务**：支持定时任务（如每日模型微调），通过 `@app.function` 装饰器定义，自动重试失败任务。
4. **文件系统集成**：挂载云存储（如 S3），处理 100GB+ 数据集，延迟 < 50ms。
5. **Python 原生**：直接运行 Python 代码，无需 Docker，支持 PyTorch、TensorFlow 等库。

### 定价
付费（按用量计费，免费层含 $30/月额度），GPU 起价 $0.0002/秒（CPU 任务）。

### 优点
1. **零运维**：自动管理 GPU 启动和缩放，适合不熟悉 DevOps 的团队。
2. **成本可控**：按秒计费，闲置时无成本，适合间歇性任务（如周末推理）。
3. **快速启动**：冷启动延迟 < 1 秒，比传统云服务器快 10 倍。

### 缺点
1. **供应商锁定**：依赖 Modal 平台，迁移到其他云服务需重写代码。
2. **资源限制**：单次任务最长运行 24 小时，不适合长期训练任务。

### 适用场景
按需推理和实验，如原型验证、数据科学竞赛、学术研究，适合预算有限且需快速迭代的团队。
