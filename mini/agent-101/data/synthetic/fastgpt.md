# FastGPT 评测

## FastGPT

### 简介
FastGPT 是一个开源的知识库问答系统，专注于高效检索和生成（RAG）。它基于向量数据库和 GPT 模型，支持 PDF、图片等文档解析，适合需要私有知识库的企业。

### 核心功能
- **文档处理**：自动解析 PDF、Word、图片（OCR），提取文本并分段，处理速度达 100 页/分钟。
- **向量检索**：使用 Milvus 或 Pinecone 作为后端，搜索延迟低于 200 毫秒，准确率 90%+。
- **自定义提示词**：可调整回答风格，例如设置“用简洁语言回复”或“引用来源”。
- **API 集成**：提供 RESTful API，方便嵌入现有系统，如 CRM 或网站。

### 定价
- **Free**：开源版本，完全免费，需自部署（推荐 4 GB RAM + 1 vCPU）。
- **Freemium**：云服务版免费层（100 次查询/天），付费版 $19.99/月（10,000 次查询，含托管）。
- **Paid**：企业版 $99/月，支持私有部署和 SLA 保障。

### 优点
- 检索精准：向量搜索结合重排序，减少幻觉，例如回答法律问题时引用具体条款。
- 部署灵活：支持 Docker 和 Kubernetes，适合内网环境。
- 社区支持：GitHub 10,000+ star，文档详细，更新频率高。

### 缺点
- 硬件要求高：大规模知识库需要 GPU 或大内存，自部署成本不低。
- UI 较简陋：管理界面功能有限，缺乏仪表盘和高级分析。

### 适用场景
适合内部知识管理，如企业用 FastGPT 构建员工 FAQ 系统，或教育机构快速回答学生问题。
